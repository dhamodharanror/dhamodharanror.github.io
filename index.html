Builds

Things I've designed and shipped at CultureMonkey.

CultureMonkey × Slack
Survey delivery and completion directly inside Slack workspaces.
Integrations Rails
▾
Built the full Slack bot integration — OAuth app install flow, survey delivery as interactive Slack messages, and response capture back into the platform. Participation rates improved significantly when surveys reached employees where they already work. This is now a core product feature.
Microsoft Teams Integration
Adaptive card surveys and bot installation across Teams tenants.
Integrations MS Teams
▾
Designed and built the Teams bot — adaptive card survey delivery, tenant-level bot installation, and a retry/failure handling layer for message delivery. Handled the Microsoft OAuth flow and proactive message routing to users across multiple tenants.
HRIS Integration Ecosystem
BambooHR, Workday, Keka, Darwinbox — employee data in sync.
Integrations HRIS
▾
Built a provider factory pattern that handles 10+ HRIS connectors — BambooHR, Workday, Keka, Darwinbox, ZohoPeople, FreshService, and more. Each connector handles first-sync, incremental sync, employee hierarchy mapping, and team structure. Adding a new HRIS doesn't touch existing ones.
Dual Analytics Pipeline
Elasticsearch for real-time search. ClickHouse for OLAP at scale.
Infrastructure Analytics
▾
Designed the migration from a pure Elasticsearch analytics setup to a dual-pipeline architecture. Elasticsearch continues to handle real-time search and segment filtering. ClickHouse ingests every 5 minutes for heavy aggregations — response trends, benchmark comparisons, cross-tenant analytics. Query time on large reports dropped dramatically.
Multi-Tenant Survey Platform
Schema-per-tenant PostgreSQL — complete data isolation at scale.
Systems Multi-tenancy
▾
Every tenant gets its own isolated PostgreSQL schema via ros-apartment. Subdomain middleware switches tenant context per request. Background jobs carry tenant context automatically — no manual switching. Survey lifecycle (draft → publish → run → end), participant flows, autosave, and submission all operate within this model with zero cross-tenant data exposure.
SAML SSO + 2FA Auth Stack
Enterprise-grade auth — SP/IdP SAML, Microsoft OAuth, TOTP.
Systems Auth
▾
Built the full authentication layer — SAML 2.0 SP/IdP-initiated SSO with Google and custom IdPs, Microsoft OAuth for email sign-in, TOTP-based 2FA via ROTP, CanCanCan role-based access control, and a separate CM Internal SAML flow. Tenant-aware routing throughout.
